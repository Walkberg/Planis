# Placez vos icônes ici

Pour une meilleure compatibilité multi-plateforme, ajoutez :

- **icon.ico** - Icône Windows (256x256 ou multi-résolution)
- **icon.icns** - Icône macOS (généré avec iconutil)
- **icon.png** - Icône Linux (512x512 recommandé)

## Génération des icônes

Vous pouvez utiliser des outils comme :
- [electron-icon-maker](https://www.npmjs.com/package/electron-icon-maker)
- [Photoshop](https://helpx.adobe.com/photoshop/using/create-icons-windows.html)
- En ligne : [CloudConvert](https://cloudconvert.com/)

## Icône temporaire

Pour le développement, Electron utilisera l'icône par défaut jusqu'à ce que vous ajoutiez vos propres icônes.
